Сборка через Maven
==================

Установка и настройка
---------------------
* Загрузить: http://maven.apache.org/download.cgi

Центральный репозиторий Maven
-----------------------------
* Поискать артефакт: http://search.maven.org/

Свои репозитории
----------------
* Локальный репозиторий: %USERPROFILE%\.m2 

Консольный запуск
-----------------
```
mvn
```
